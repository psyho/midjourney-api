# Midjourney wrapper

A simple script that interacts with Midjourney Bot on Discord and downloads the generated images.

## Usage

Set the ENV vars in `.env`. Then, generate the images with:

```shell
# ./bin/imagine <prompt>
$ ./bin/imagine a great success --ar 16:9 --stylize 1000
```
